# Python에서 계산하기

## 기본적인 사칙연산

- 더하기 (+)
- 빼기 (-)
- 곱하기 (*)
- 제곱 (**)
- 나누기
  1. 일반적인 나누기 : /
  2. 몫만 구하기 : //
  3. 나머지 구하기 : %
  4. 한번에 구하기 : divmod
    ```python
    10 / 3 = 3.33333..
    10 // 3 = 3
    10 % 3 = 1
    divmod(10,3) = (3,1)    
    ```
