# 복사
## 1. 할당
 - 같은 할당 주소를 공유
   ```python
   a = [1, 2, 3]
   b = a
   b[0] = 100

   print(a) # [100, 2, 3]
   print(b) # [100, 2, 3]
   print(id(a), id(b)) # 둘이 같은 메모리 주소를 가지고 있음을 확인할 수 있다.
   ```

## 2. 얕은 복사
 - **슬라이싱**을 통해 생성된 객체는 원본 객체와 독립적으로 존재
   ```python
   a = [1, 2, 3]
   b = a[:]
   b[0] = 100

   print(a) # [1, 2, 3]
   print(b) # [100, 2, 3]
   ```
 - **copy**
    ```python
    a = [1, 2, 3]
    c = a.copy()
    c[0] = 100

    print(a) # [1, 2, 3]
    print(c) # [100, 2, 3]
    ```
 - 얕은 복사의 한계
    ```python
    a = [1, 2, [1, 2]]
    b = a[:]
    b[2][0] = 999
    
    print(a) # [1, 2, [999, 2]]
    print(b) # [1, 2, [999, 2]]

    c = a.copy()
    c[2][0] = 999

    print(a) # [1, 2, [999, 2]]
    print(c) # [1, 2, [999, 2]]
    ```
## 3. 깊은 복사
- import copy 필수
- 메모리 주소가 다른 복사본이 생긴다.
   ```python
   import copy

    a = [1, 2, [1, 2]]
    b = copy.deepcopy(a)
    b[2][0] = 999

    print(a) # [1, 2, [1, 2]]
    print(b) # [1, 2, [999, 2]]
   ```
