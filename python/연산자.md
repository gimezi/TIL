# 연산자

## 복합 연산자
연산과 할당이 함께 이뤄짐
|기호| | |
|---|---|---|
|+=|a += b|a = a + b|
|-=|a -= b|a = a - b|
|*=|a *= b|a = a * b|
|/=|a /= b|a = a / b|
|//=|a //= b|a = a // b|
|%=|a %= b|a = a % b|
|**=|a **= b|a = a ** b|

## 비교 연산자
|기호|내용|
|---|---|
== |같다 
!= |같지 않음
is |같다
is not | 같지않음

### '=='와 'is'의 차이
 -> '=='는 동등성 'is'는 식별성
 
 동등성은 값이 같은지 비교하는 거, 식별정은 레퍼런스(주소)가 같은지 비교

## 논리 연산자
- **and** : 논리곱, 둘다 True여야 True
- **or** : 논리합, 둘 중 하나만이라도 True이면 True
- **not** : 논리부정, 단일 피연산자를 부정
